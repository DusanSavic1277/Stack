\# Custom Stack Implementation in Java



This project demonstrates a custom implementation of the Stack data structure

using Java, without relying on java.util.Stack.



\## Features

\- Generic Stack (MyStack<T>)

\- LIFO behavior

\- Dynamic array resizing

\- push, pop, peek, isEmpty, size methods

\- Proper exception handling



\## Example

```java

MyStack<Integer> stack = new MyStack<>();

stack.push(10);

stack.push(20);

stack.pop();
